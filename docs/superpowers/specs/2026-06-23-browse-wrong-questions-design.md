# 错题浏览功能 — 设计文档

> 日期：2026-06-23
> 状态：已确认

## 目标

新增一个独立的「错题本」页面，让用户按六科（语文/数学/英语/物理/化学/生物）自由翻阅全部错题，随时自测是否还记得，标记结果同步更新掌握状态。

## 与现有功能的关系

- **保留** `/review/[subjectId]` 间隔重复复习页不变
- **保留** `/subject/[id]` 科目知识树页不变
- **新增** `/browse` 独立错题浏览页
- 自测提交复用 `POST /api/review/submit` 接口，与复习页共享同一套 mastery_level 更新逻辑

## 路由设计

| 路由 | 页面 | 说明 |
|------|------|------|
| `/browse` | 错题浏览页 | 六科切换 + 卡片网格 |

无需新增后端接口，全部复用现有 API：

| API | 用途 |
|-----|------|
| `GET /api/questions?subject_id=X` | 加载某科全部错题 |
| `POST /api/review/submit` | 自测标记后提交结果 |

## 页面布局

```
┌─────────────────────────────────────────┐
│  Navbar (新增「错题本」入口)              │
├─────────────────────────────────────────┤
│                                         │
│  语文 │ 数学 │ 英语 │ 物理 │ 化学 │ 生物     ← 科目标签栏
│                                         │
│  ┌──────────┐  ┌──────────┐            │
│  │ 题目摘要   │  │ 题目摘要   │            │
│  │ 题型标签   │  │ 题型标签   │            │  ← 2列卡片网格
│  │ 掌握状态   │  │ 掌握状态   │            │
│  │ [查看答案] │  │ [查看答案] │            │
│  └──────────┘  └──────────┘            │
```

- 标签栏：六科目平铺，选中态高亮
- 卡片网格：响应式 1-2 列，每个卡片展示题目内容摘要 + 题型标签 + mastery 状态徽标
- 点击「查看答案」→ 卡片就地展开，底部出现正确答案/解析 + 「记得」「不记得」按钮
- 标记后卡片状态即时更新，不跳页

## 组件拆分

1. **`/app/browse/page.tsx`** — 页面主组件
   - 管理当前选中科目 state
   - 调用 `GET /api/questions?subject_id=X` 加载数据
   - 渲染标签栏 + 卡片网格

2. **`/components/BrowseCard.tsx`** — 错题卡片组件
   - 折叠/展开状态
   - 显示题目内容、题型、掌握级别
   - 展开后显示答案 + 解析 + 自测按钮
   - 调用 `POST /api/review/submit` 提交结果

3. **Navbar 修改** — 添加「错题本」导航链接

## 数据流

```
用户选择科目 → GET /api/questions?subject_id=X
              → 渲染卡片网格
用户点击「查看答案」→ 卡片展开，显示答案
用户点击「记得/不记得」→ POST /api/review/submit
                       → 卡片 mastery 标签即时更新
```

## 状态展示

| mastery_level | 显示 |
|---------------|------|
| new | 🔴 新录入 |
| learning | 🟡 复习中 |
| mastered | 🟢 已掌握 |
