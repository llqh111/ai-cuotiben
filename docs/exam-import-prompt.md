# 试卷错题分析 → AI 错题本导入提示词

> 用法：将试卷内容 + 你的错答情况贴给 Claude（网页版或 Claude Code），Claude 按此模板输出结构化 JSON，然后用 `scripts/import_json.py` 一键导入。

---

## 复制以下内容发给 Claude

````markdown
你是一位高考理科辅导老师。我发你一份试卷和我的作答情况，请帮我分析错题，输出结构化 JSON 以便导入我的错题本系统。

## 输出格式

严格输出一个 JSON 数组（包裹在 ```json 代码块中），每条错题一个对象：

```json
[
  {
    "subject_id": 2,
    "knowledge_point_name": "导数与单调性",
    "question_pattern_name": "含参求单调区间",
    "question_content": "已知 f(x)=x²-ax 在 [1,3] 递减，求 a 的取值范围。",
    "question_type": "essay",
    "correct_answer": "a ≥ 6",
    "student_answer": "a > 3",
    "solution_steps": "由 f'(x)=2x-a≤0 在 [1,3] 恒成立 → a≥2x 在 [1,3] 恒成立 → a≥6。",
    "error_analysis": "把递减条件 f'(x)≤0 写成了 f'(x)<0，漏了等号。含参恒成立用参变分离时端点取等问题没注意。",
    "improvement_tips": "单调性含参问题主动标等号条件；参变分离后验证端点。"
  }
]
```

### 字段说明

| 字段 | 必填 | 取值 |
|------|------|------|
| subject_id | ✅ | 1语文 2数学 3英语 4物理 5化学 6生物 |
| knowledge_point_name | ✅ | 从下方「知识点参考表」选取，优先精确匹配 |
| question_pattern_name | 选 | 题型名称，如「含参求单调区间」「电学实验设计」 |
| question_content | ✅ | 完整题干 |
| question_type | 选 | essay / choice / fill_blank，默认 essay |
| correct_answer | 选 | 正确答案或解析 |
| student_answer | 选 | 你的错误作答 |
| solution_steps | 选 | 分步解题思路 |
| error_analysis | 选 | 错因分析（是概念不清/计算失误/审题问题/方法选错？） |
| improvement_tips | 选 | 改进建议 |

## 使用流程

1. **拍照/输入试卷**：把试卷内容和你的作答发给 Claude（网页 / Claude Code 均可）
2. **复制 JSON**：Claude 返回分析结果，复制 ```json 代码块内容
3. **保存文件**：粘贴到本地文件，如 `math-week6.json`
4. **导入**：运行 `cd ai-cuotiben-api && python scripts/import_json.py math-week6.json`
5. **验证**：打开 http://localhost:3000/browse 查看新导入的错题

## 导入脚本用法

```bash
# 基本用法（默认连 localhost:8000）
python scripts/import_json.py result.json

# 指定后端地址
python scripts/import_json.py result.json --api http://localhost:8000
```
